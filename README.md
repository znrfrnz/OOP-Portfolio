# OOP Portfolio — Franz Emmanuel F. Baes

A personal portfolio website built as a midterm requirement for my Object-Oriented Programming class. It showcases the activities, seatwork, quizzes, and midterm exam I completed throughout the course, along with code previews and reflections for each output.

**Live site:** [baes-oop-portfolio.vercel.app](https://baes-oop-portfolio.vercel.app)

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
